pre 태그는 공백 여러개를 인식
줄 바꿈도 여러개 인식 함 

1. b는 글자를 굵게 해주는 것
2. i는 글자 기울임
3. h1-h6 숫자가 작을수록 크기가 크다
4. br은 줄 바꿈
5. pre는 공백, 줄바꿈을 자유롭게 쓸 수 있음
6. hr은 수평선이다. (줄)
7. ul은 순서가 없는 리스트 + li 리스트
8. ol은 순서가 있는 리스트 + li 리스트
9. a href = "" 는 하이퍼링크
10. img scr = "" 이미지를 넣는 명령어 --> width/ height 을 %로 넣을 경우, 화면 비율에 맞게 조정된다.
10_2. alt는 그림을 로드하지 못 했을 때 뜨는 설명 지정
11. table - tr - td 행열을 가지고 있는 표를 지정
11_2. caption 은 표 위에 제목 지정
11_3. border는 표를 감싸는 선 지정
11_4. rowspan 행 병합 / colspan 열 병합
11_5. th는 표 안에 설명.
11_6. style은 직접 스타일을 적용. 보통은 css파일을 따로 만든다.
